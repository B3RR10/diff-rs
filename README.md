## Getting Started

Install it in different ways:

**actually not availible**
```
cargo install diff_rs
```

or directly over github:

```
git clone https://github.com/miguelberrio91/diff.rs
cd diff.rs
make install
```

## Usage

**Git diff**
```
$ git diff | diff_rs
```
